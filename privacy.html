♨ プライバシーポリシー

最終更新日：2025年6月21日

WhereSentou（以下「本アプリ」）は、ユーザーのプライバシーを尊重し、個人情報の適切な取り扱いに努めます。本ポリシーは、本アプリが収集する情報と、その利用目的について説明します。

1. 収集する情報

位置情報（精密な位置情報）

  • 目的：現在地周辺の銭湯を検索・表示するため、およびナビゲーション機能のため
  • 利用方法：アプリ内でのみ使用し、外部には送信しません
  • 取得タイミング：アプリ使用中のみ（バックグラウンドでの常時取得は行いません）

アカウント情報

  • Apple サインインまたは Google サインインを利用した場合、表示名とメールアドレスを取得します
  • 取得した情報はFirebase Authentication に保存し、訪問記録やコミュニティ機能の識別に使用します

利用状況データ（Firebase Analytics）

  • アプリの改善を目的として、匿名化された利用状況データを収集します
  • 端末の種類、OS バージョン、クラッシュレポートなどが含まれます
  • 個人を特定する情報は含みません

ユーザーが入力したデータ

  • 銭湯の訪問記録（訪問日、銭湯名、移動距離）
  • コミュニティによる銭湯情報の追加・編集内容
  • これらは Firebase Firestore に保存されます

購入情報

  • プレミアムサブスクリプションの購入状況は Apple の App Store を通じて管理されます
  • 本アプリはクレジットカード番号等の決済情報を直接取得・保存しません

2. 情報の利用目的

  • 現在地周辺の銭湯の検索・表示・ナビゲーション
  • 訪問記録の保存と複数端末間での同期
  • コミュニティ機能（銭湯の追加・編集・ランキング）の提供
  • サブスクリプション機能の管理
  • アプリの品質向上・バグ修正

3. 情報の第三者提供

収集した個人情報は、以下の場合を除き第三者に提供しません。

  • ユーザーの同意がある場合
  • 法令に基づく開示要求がある場合

本アプリは以下のサービスを利用しており、各サービスのプライバシーポリシーが適用されます。

  • Google Firebase（Firebase Authentication / Firestore / Analytics）
  • Google Sign-In
  • OpenStreetMap / Overpass API

4. データの保存と管理

収集したデータは Google Cloud（Firebase）のサーバーに保存されます。データは適切なセキュリティ対策のもとで管理されます。アカウントを削除した場合は、関連する個人データも合わせて削除します。削除をご希望の場合は下記お問い合わせ先までご連絡ください。

5. 子どものプライバシー

本アプリは13歳未満の子どもを対象としておらず、意図的に13歳未満の個人情報を収集しません。

6. プライバシーポリシーの変更

本ポリシーは予告なく変更することがあります。重要な変更がある場合はアプリ内またはこのページでお知らせします。
